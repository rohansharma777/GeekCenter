About GeekCenter

GeekCenter is a web application made using django-a python frame work. We the students at GEC Bhavnagar have created a Blog Generator and a demo of IOT i.e a PC tracker using GPS API and Browser GEOLOCATION .

Idea:

The main idea is to:

  • Make the site simple organized and responsive
  • Presently the site gives location of a PC using it's IP address
  • The future plan is to locate servers on the map and get logs from each and every server
  • The logs can then be accessed by various users
  • The idea of server logs maintainence is used by several companies (yes facebook and instagram or any company you name)
  • This technology is used for recovering from failures using logs and for storing all kernel related information and system(here server) health information
  • The site also has a blog/forum called BlogGen which is a Blog generator like Blogspot from google
  • The BlogGen has functionality like :
    1. Writing post on the Go
    2. Editing post on the Go
